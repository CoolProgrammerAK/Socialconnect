## Socialconnect
Socialconnect-  Connect with friends and the world around you

## Features

* Technology ued
  * React (create-react-app)
  * Node-js(Express)
  * Mongodb
  * Cloudinary
  * Material-ui
  
* Features:
  * Sign In
  * Sign Up
  * Sign Out
  * Protected Routes with Authorization
  * Post video or photo
  * Following / Followers feature
  * Real time Notification feature
  * Chat feature
  * Online friends feature
